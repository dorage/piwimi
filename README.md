## Install

    $ git clone https://github.com/dorage/nodepsychotest.git
    $ cd nodepsychotest
    $ npm install

## Configure app

.gitignore 에 .env.development를 추가해주세요.

    $ npm i

## Running the project

NODE_ENV = 'development' 으로 실행됩니다.

    $ npm start

NODE_ENV = 'production' 으로 실행됩니다.

    .env.production 이 있어야합니다.
    $ npm run production
